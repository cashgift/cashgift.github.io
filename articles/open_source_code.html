  • 开源代码

    在份子记账iOS版开发过程中，这些遵循MIT license开源协议的库给与了巨大的帮助，在此非常感谢作者的贡献

    # RealmSwift

    数据库，就是它构建了你的人情数据结构，稳定、好用。

    https://github.com/realm/realm-cocoa (opens new window)

    # SnapKit

    超级好用的布局库，谁用谁知道，是它让份子记账绘制出美丽的页面。

    https://github.com/SnapKit/SnapKit (opens new window)

    # AAInfographics

    各种炫酷统计图工具库，没有它我是画不出来你的统计数据图形的。

    https://github.com/AAChartModel/AAChartKit-Swift (opens new window)

    # SwiftDate

    一个日期操作库，帮你精准记录人情时间。

    https://github.com/malcommac/SwiftDate (opens new window)
